# E-Commerce Project

This repository contains the source code for the E-Commerce project.

## Project Structure

All project files are located inside the `ecommerce_proj` directory:
- `ecommerce-backend-springboot-main/`: Spring Boot backend source code.
- `Thuc_tap_co_so-main/`: Frontend source code.
- `Machine_Learning-E_Commerce-main/`: Machine learning models and analysis scripts.
